# AMU Surveillance

Surveillance of [[Antimicrobial Use (AMU)]] is well established in the Nordic countries, with nationwide systems tracking consumption in both humans and animals. See the [[Antimicrobial Use (AMU)]] concept page for detailed country-by-country data sources and responsible entities.

## Key Data Sources

### Human AMU
- **Prescription databases** - capture outpatient dispensing data
- **Wholesale data** - track supplies to hospitals and institutions
- **Patient records** - hospital medication use
- Standardised using **ATC/DDD** system (WHO Collaborating Center at [[NIPH]])

### Veterinary AMU
- **Sales data** from pharmaceutical wholesalers
- **Prescription registers** (e.g., VetStat in [[Denmark]], VetReg in [[Norway]])
- Species-level and indication-level data in some countries

### Reporting
- Human AMU → **ESAC-Net**
- Veterinary AMU → **ESVAC / ESUAvet** (renamed 2023)
